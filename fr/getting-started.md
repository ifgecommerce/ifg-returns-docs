# Prise en main — IFG Returns & Withdrawals

[🇬🇧 English](../getting-started.html) · [🇮🇹 Italiano](../it/getting-started.html) · [🇩🇪 Deutsch](../de/getting-started.html) · 🇫🇷 **Français** · [🇪🇸 Español](../es/getting-started.html)

Trois étapes courtes : configurez le formulaire de retour sur votre boutique en ligne, découvrez ce que voit un client, puis gérez une demande depuis votre interface d'administration. La plupart des boutiques sont entièrement configurées en moins de dix minutes.

## 1. Configurer le formulaire de retour

**1. Terminez la configuration guidée.**
Ouvrez l'application et renseignez les informations légales de votre entreprise ainsi que votre adresse de retour. Ces informations sont obligatoires : ce sont elles que vos clients voient sur leur confirmation de rétractation, et c'est là que les articles retournés sont envoyés ; l'application les suit également dans le cadre de votre statut de conformité.

**2. Ajoutez un bloc de retour à votre thème.**
Utilisez le raccourci « Ouvrir l'éditeur de thème » à la fin de la configuration guidée (ou accédez-y directement depuis votre interface d'administration Shopify). Cliquez sur *Ajouter un bloc* / *Intégrations d'application*, recherchez **IFG Returns**, et choisissez le bloc adapté à l'emplacement souhaité :

- `Module` — intègre le formulaire complet dans une page, comme une page dédiée « Retours ».
- `Pop-up` — un bouton qui ouvre le formulaire dans une fenêtre modale, à placer n'importe où sur une page.
- `Floating` — un bouton flottant visible sur toutes les pages ; activé depuis le panneau *Intégrations d'application* plutôt qu'ajouté à une seule page.

> Vous pouvez en activer plusieurs à la fois — par exemple un bouton flottant sur tout le site, associé à une page dédiée pour un lien dans le pied de page.

**3. Enregistrez le thème.**
Le tableau de bord de l'application détecte automatiquement quel bloc est actif — aucune confirmation manuelle n'est nécessaire.

## 2. Ce que voit votre client

**1. Aucun compte nécessaire.**
Les clients ouvrent le formulaire de retour directement depuis votre boutique en ligne — aucune connexion requise. La langue du formulaire suit automatiquement la langue configurée de votre boutique, y compris la mise en page complète de droite à gauche pour l'arabe et l'hébreu.

**2. Recherche de commande.**
Le client saisit son numéro de commande, son e-mail et la date de la commande. Ces informations sont vérifiées en temps réel par rapport à la commande réelle dans Shopify, de sorte que seules les commandes authentiques peuvent être soumises.

**3. Confirmation instantanée.**
Une fois la demande envoyée, le client reçoit une confirmation immédiate avec la référence de son dossier — la demande apparaît désormais dans votre interface d'administration.

## 3. Gérer une demande

**1. Ouvrez « Retours » dans l'application.**
Chaque nouvelle demande apparaît ici avec un badge de statut (en attente, approuvé, remboursé, etc.) afin que vous puissiez voir en un coup d'œil ce qui nécessite votre attention.

**2. Examinez et approuvez.**
Ouvrez un dossier, vérifiez les articles et le motif, puis cliquez sur **Approuver**. Une fois l'article retourné reçu, cliquez sur **Confirmer la réception**.

**3. Remboursez.**
Sélectionnez les articles et le montant à rembourser, puis confirmez. Sur les offres incluant les remboursements directs, le remboursement est émis immédiatement depuis l'application — sans avoir à basculer vers la page de commande Shopify séparée.
